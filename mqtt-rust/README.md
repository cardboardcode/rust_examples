## **What Is This?**
This is a minimal example of creating a Message Queueing Telemetry Transport (MQTT) Publisher and Subscriber. 

## **Build** :hamme:

```bash
cargo build
```

## **Run**

```bash
./target/debug/syncpubsub
```

```bash
./target/debug/asyncpubsub
```
